# ИГУ Колледж

React + Vite сайт колледжа с Apple-like glass интерфейсом, React Router, Tailwind CSS, RU/KY переключателем и защищенной серверной админ-панелью.

## Запуск

```bash
npm install
npm run dev
```

Клиент откроется на `http://127.0.0.1:5173`, API работает на `http://127.0.0.1:4000`.

## Админ-панель

Админка не отображается в меню. Вход: `http://127.0.0.1:5173/admin-login`.

Админка редактирует только новости, документы, галерею и состав ПЦК. Фото, документы и видео загружаются в `server/uploads` через защищенный API.

Создайте `.env` по примеру `.env.example` и задайте сильный пароль:

```bash
ADMIN_LOGIN=admin
ADMIN_PASSWORD=change-this-strong-password
PORT=4000
```

Если `.env` не создан, сервер использует dev-доступ `admin / admin123`.

Лимиты загрузки: фото до 10 МБ, документы до 25 МБ, видео до 50 МБ.
